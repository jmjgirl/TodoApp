# Solo projcet : To-do App
단순한 CRUD 기능을 기본으로 구성  


## 프로젝트 소개  
  
### 프로젝트 개요
- API Docs와 테이블 구성을 참고하여 To-Do App 개발
- **Tomcat**을 이용하여 로컬 환경에서 수동 배포
- **Ngrok**을 통해 외부에서 접근 가능한 로컬 환경 구축
- Postman과 http://todobackend.com/ 사이트 이용하여 결과 확인
